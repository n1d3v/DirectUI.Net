DirectUI.Net is a library for C# (.NET) that allows you to use DirectUI in C# projects.
# Why would you make this?
This project was specifically made because Windows Forms is extremely limited in terms of making native UIs. You can make some good ones, sure, but you don't have that fluid experience that you get with Windows Presentation Framework (WPF).

DirectUI was never released by Microsoft, despite being a fully functional library that Microsoft developers used to make Control Panel applets, various Microsoft products (i.e. Windows Live Messenger) and Windows components.
